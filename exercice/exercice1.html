Introduction to HTML

HTML (hypertext Markup Language) is used to created document on the World Wide Web. It is simply a collection of certain key words called "Tags" that are helpful in writting the document to be displayed using a browser on internet. HTML was developed by Tim Berners-Lee in 1992.
